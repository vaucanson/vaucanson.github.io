Procédure pour forker un dépôt dans l'intention d'y contribuer

* Dans Github, forker le dépôt qui nous intéresse
* Dans Netbeans créer un projet BOUM_DAY_JAVA
* dans la console, aller dans le dossier NetbeansProject/BOUM_DAY_JAVA
* $ rm -rf *
* $ git init
* $ git remote add origin https://[mon fork sur github]
* $ git remote add upstream https://[le dépôt d'origine]
* $ git config remote.pushdefault origin
* $ git config push.default current
* $ git pull upstream master

Pour contribuer via une pull request

1) Faire mes modifs
2) Commiter mes modifs :
$ git add
$ git commit -m 'mon commentaire'
3) Pousser sur Origine :
$ git push
4) Dans la page github d'Origine, cliquer sur « pull request »
