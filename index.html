Forker un dépôt dans le but d'y contribuer

1) Forker un dépôt dans github : dans la page du dépôt, cliquer sur "fork" en haut à droite. À partir de maintenant j'appellerai Origine notre fork, et Upstream le dépôt d'origine.
2) Cloner Origine en local :
git clone https://github.com/
3) Configurer le dépôt local pour qu'il pousse par défaut sur Origine et
git config remote.pushdefault origin # Publier par défaut sur le dépôt 'origin'
git config push.default current # Publier par défaut sur une branche portant le même nom que la branche courante dans le dépôt de publication
4) Ajouter le remote Upstream, puis le fetcher :
git remote add upstream https://github.cm/
git fetch uptsream

Pour contribuer via une pull request

1) Faire mes modifs
2) Commiter mes modifs :
git add
git commit -m 'mon commentaire'
3) Pousser sur Origine :
git push
4) Dans la page github d'Origine, cliquer sur « pull request »
